skink CI Server
===============

skink is a Continuous Integration server. It is as simple as it gets. Have a git repository? A build command? That's enough to get started.

Check the :doc:`quickstart` page to get up and running.

If you are trying to integrate with skink, you can check the :doc:`plugin_authoring` page or the :doc:`skink_events` page to get more info on how to interact with skink.
